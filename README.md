# 🏥 Patient Management System (General Clinic)

A modern, full-stack Patient Management System designed to streamline clinic workflows, automate prescription generation, and manage medical inventory. The application features a beautiful, responsive fluid design with a clean glassmorphism aesthetic.

---

## ✨ Key Features

- **📊 Comprehensive Dashboard:** Instantly view clinic statistics including total patients, appointments, medications, and inventory metrics at a glance.
- **👤 Patient Onboarding & Records:** Manage full patient profiles including secure IDs, medical history (previous operations/treatments), symptoms, and family background.
- **📅 Dynamic Appointment Tracking:** Log and update appointments with integrated fields for diagnosis, required tests, lab/X-ray results, and custom physician notes.
- **💊 Medication Database:** A dedicated panel to add, view, and organize available drugs, active ingredients, and specific usage notes.
- **📋 Smart Prescription Generator:** Automatically generates high-fidelity, printable medical prescriptions complete with dynamic tables and a professional layout (Bilingual English/Arabic support).
- **📦 Clinic Inventory Management:** Real-time stock tracking for clinic supplies with automated status indicators (In Stock, Low Stock, Out of Stock).

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS (Custom glassmorphic utility styles)
- **Backend/Logic:** Vanilla JavaScript state management & PHP backend API architecture
- **Design Paradigm:** High-fidelity UI/UX, responsive layouts, pixel-perfect component spacing
